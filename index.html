상품 목록 페이지

: 카페에서 판매하는 상품들을 볼 수 있는 페이지를 3가지 카테고리(음료, 디저트, 상품)으로 나누어 상품이 등록된 시간 순으로 정렬했습니다.

    [기능설명]
  • 상품목록 : 카테고리 선택에 따라 DB에서 가져온 상품 목록을 화면에 출력합니다.
  • 상품 자세히보기 : 상품의 이미지를 클릭하거나 자세히보기를 클릭하면 상세페이지로 이동합니다.
  • 상품을 장바구니에 담기 : 해당상품 1개를 장바구니에 담습니다.

상품 자세히보기

: 상품을 자세히보기 했을 때 나오는 화면입니다.

    [기능설명]
  • 자세히 보기 : 상품1개에 대한 정보를 DB에서 가져와서 보여줍니다.
  • 장바구니 담기 : 상품 갯수를 선택해서 장바구니에 담을 수 있습니다.
  • 추천메뉴 : 세가지 카테고리에 있는 상품을 랜덤으로 3개를 추천합니다. 각 제품을 상세보기, 장바구니에 담기 할 수 있습니다.

문의게시판

: 고객들이 문의를 하고 관리자페이지에서 답글작성을 하는 문의게시판입니다.

    [기능설명]
  • 문의목록 : 회원이 작성한 문의글 목록을 DB에서 가져와서 출력합니다.
  • 문의글 작성 : 문의하기 버튼을 누르면 문의를 작성하는 페이지로 이동합니다. 회원만 이동이 가능하도록 유효성을 부여했습니다.
  • 문의글 자세히보기 : 문의글의 제목을 누르면 문의글 상세로 이동합니다.
  • 검색기능 : 제목, 내용, 제목+내용 검색의 결과에 따라 문의 목록을 출력할 수 있습니다.
