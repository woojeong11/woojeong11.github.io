Cafe Weather 홈페이지

[프로젝트 설명]
Spring Framework를 사용하여 처음으로 팀 프로젝트로 진행한 카페 홈페이지 프로젝트입니다. 가상의 카페 홈페이지로써 관리자인 점주가 있고 카페를 이용하는 일반 고객이 존재합니다.
기본적인 기능으로는 날씨에 따라 메뉴추천, 메뉴설명, 주문, 결제,
리뷰작성, 실시간 채팅 기능이 있습니다.

README

📌 Cafe Weather 프로젝트에서 제가 맡은 기능입니다.


💡 판매-상품 목록 페이지

    카페에서 판매하는 상품들을 3가지 카테고리(음료, 디저트, 상품)로 나누어 상품이 등록된 시간 순으로 정렬했습니다. 각 카테고리 클릭 시 해당 상품목록이 출력됩니다.


[기능설명]
  • 상품목록 : 카테고리 선택에 따라 DB에서 가져온 상품 목록을 화면에 출력합니다.
  • 상품 자세히보기 : 상품의 이미지를 클릭하거나 자세히보기를 클릭하면 상세페이지로 이동합니다.
  • 상품을 장바구니에 담기 : 해당상품 1개를 장바구니에 담습니다. 장바구니 목록에 같은 상품이 있으면 수량이 더해집니다.

    • [화면]


    💡 상품 자세히보기

        상품을 자세히보기 했을 때 나오는 화면입니다.


    [기능설명]
  • 자세히 보기 : 상품1개에 대한 정보를 DB에서 가져와서 보여줍니다.
  • 장바구니 담기 : 상품 갯수를 선택해서 장바구니에 담을 수 있습니다.
  • 추천메뉴 : 세가지 카테고리에 있는 상품을 랜덤으로 3개를 추천합니다. 각 제품을 상세보기, 장바구니에 담기 할 수 있습니다.

    • [화면]


    💡 문의게시판

        고객들이 문의를 하고 관리자페이지에서 답글작성을 하는 문의게시판입니다.


    [기능설명]
  • 문의목록 : 회원이 작성한 문의글 목록을 DB에서 가져와서 등록일 기준 내림차순으로 정렬하여 출력합니다. 관리자가 원글에 답글을 달면 원글 밑에 정렬되어 출력됩니다.
  • 문의글 작성 : 문의하기 버튼을 누르면 문의를 작성하는 페이지로 이동합니다. 회원만 이동이 가능하도록 유효성을 부여했습니다.
  • 문의글 자세히보기 : 문의글의 제목을 누르면 문의글 상세로 이동합니다.
  • 검색기능 : 제목, 내용, 제목+내용 검색의 결과에 따라 문의 목록을 출력할 수 있습니다.
  • 문의작성 및 등록 : 회원이 제목, 내용, 이미지를 넣고 비밀글 여부를 선택한 뒤 작성하면 문의글이 DB에 저장되고 목록에 출력됩니다.

    • [화면]


    💡 장바구니

        상품목록이나 상품상세에서 장바구니담기를 눌렀을 때 장바구니에 해당 상품이 저장됩니다. 추가할때마다 장바구니에 목록이 추가되고, 해당하는 상품이 중복이면 수량만 추가됩니다.


    [기능설명]
  • 장바구니 목록 : 회원 또는 비회원이 담은 장바구니 목록을 보여줍니다.
  • 수량 수정 : 수량을 조절하면 자동으로 총 갯수와 총 금액이 실시간 변경되며 주문하기를 누르면 수량이 수정되어 DB에 저장됩니다.
  • 삭제, 선택삭제, 장바구니 비우기 : 여러가지 삭제 기능을 부여했습니다.

    • [화면]


    💡 주문서작성

        배송받을 배송지정보를 입력하는 주문서를 작성합니다. 이전페이지인 장바구니에서 수량변경이 되었으면 반영되도록 했고 총 수량과 총 결제예정액을 보여줍니다.


    [기능설명]
  • 상품관리 : 상품을 카테고리 별로 등록이 가능합니다. CRUD, 사진을 여러장 추가할 수 있는 기능이 있습니다.
  • 문의게시판관리 : 회원이 문의한 게시글에 답글을 관리자페이지에서 작성할 수 있습니다. 비밀글 여부는 회원이 작성한 원글의 value에 따라 달라집니다.

    • [화면]


    💡 관리자페이지

        상품을 등록, 삭제, 관리를 할 수 있는 상품관리페이지와 문의게시판에 답글을 달 수 있는 문의게시판관리페이지 입니다.


    [기능설명]
  • 상품관리 : 상품을 카테고리 별로 등록이 가능합니다. CRUD, 사진을 여러장 추가할 수 있는 기능이 있습니다.
  • 문의게시판관리 : 회원이 문의한 게시글에 답글을 관리자페이지에서 작성할 수 있습니다. 비밀글 여부는 회원이 작성한 원글의 value에 따라 달라집니다.
